# Image Compressor Pro (PWA)

Современное web-приложение для профессионального сжатия изображений. Работает прямо в браузере (оффлайн), использует WebAssembly (WASM) для качественной обработки.

## 🚀 Быстрый Старт

Вам понадобится установленный **Node.js** (версия 18+).

1. **Скачайте и откройте папку проекта** в терминале.
2. **Установите зависимости**:
   ```bash
   npm install
   ```
3. **Запустите приложение**:
   ```bash
   npm run dev
   ```
   Откройте ссылку `http://localhost:5173` в браузере.

## ✨ Основные возможности

### 1. Профессиональное сжатие (WASM)
Приложение использует библиотеки **MozJPEG** и **libwebp** (через WebAssembly), что гарантирует:
- Высокое качество сжатия.
- Поддержку прозрачности (без черного фона).
- Полную конфиденциальность (файлы не отправляются на сервер).

### 2. Пресеты для Веб-мастеров
В разделе "Настройки" доступны готовые шаблоны:
- **Modern Web**: WebP, качество 80% (оптимально для сайтов).
- **Content**: WebP, ширина 800px (для статей и блогов).
- **Thumbnail**: WebP, ширина 600px (для миниатюр).
- **Legacy**: JPEG, качество 85% (максимальная совместимость).

### 3. Удобная очередь (Queue) и Автоматизация
- **Мгновенный старт**: Добавление файлов через Drag & Drop или кнопку моментально переводит приложение в режим сжатия без лишних кликов.
- **Сортировка**: По имени (A-Z) или размеру (от больших к меньшим).
- **Сравнение**: Визуальный слайдер "До/После" для проверки качества.
- **Подсказки**: Наведите курсор на любой элемент, чтобы узнать, что он делает.

### 4. Экспорт и скачивание
- **Умный экспорт**: 
  - Если файлов **несколько** — приложение собирает один удобный `.zip` архив.
  - Если скачивается или обрабатывался всего **1 файл** — скачивание происходит напрямую (.webp/.jpeg), экономя ваше время.
- **Сохранить в папку**: Прямое сохранение на диск через File System Access API (только для Chrome/Edge на ПК).
- **Автоочистка**: Через 1.5 секунды после успешного скачивания очередь картин автоматически очищается, и приложение само переводит вас обратно на первый экран для загрузки новой партии.

### 5. Оптимизация и Производительность
- **Многопоточность (Concurrency)**: Приложение использует мощный `WorkerPool` для одновременного сжатия множества файлов. Лимиты потоков зависят от мощности вашего процессора (hardware concurrency).
- **Облачное PWA-обновление**: Интегрирован "взрослый" подход к обновлениям (prompt mode). Приложение само слушает фон и, если разработчик выпускает новую версию, внизу экрана появляется аккуратная кнопка «Update Available 🚀» для мгновенного бесшовного обновления в один клик.

## 📱 Установка как приложение (PWA)

Вы можете установить компрессор как обычную программу на компьютер:
1. Откройте приложение в **Google Chrome** или **Edge**.
2. В адресной строке справа нажмите иконку **"Установить" (Install)**.
3. Приложение появится в меню "Пуск" или "Программы" и будет работать даже без интернета.
